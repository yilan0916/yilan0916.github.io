学习资料

极海：https://www.bilibili.com/video/BV1Rg411y7SX

IDEA快捷键

作用 Mac按键 Win按键
查看方法与变量的使用 ctrl + option + h ctrl + alt + h
查找变量的使用 option + F7 alt + F7
查看类继承关系 ctrl + h ctrl + h
查看方法实现 option + cmd + b ctrl + alt + b
查看方法中父类实现 cmd + u ctrl + u
类中的方法搜索 cmd + F12
全局搜索文件 cmd + o ctrl + n
全文搜索 cmd + shift + f ctrl + shift + f
快速生成getter/setter方法 cmd + n alt + insert

debug技巧

断点跳转(Run to Cursor)

鼠标点击代码行数

debug“撤回”

删除并重置调用栈，效果类似debug的“撤销” : debug 中terminal 点击红色x(Drop Frame)

条件debug

右键双击红色图标，输入Condition条件

例如：user.getName().equals(“土妹”)

固定表达式跟踪

Variables窗口点击”+号” New Watch，输入条件

忽略部分断点

Mac：cmd + shift + F8 忽略断点不选中

Win：ctrl + shift + F8

动态执行表达式

断点过程中右键(option+F8)选择 Evaluate Expression输入条件

多线程debug

右键点击断点，将Suspend由All修改为Thread

在JDK源码层增加信息输出

右键断点,点击More(option+cmd+F8),在Evalute and log中输入需要信息
