软件

ClassX

github下载，选择tar.gz格式
https://github.com/yichengchen/clashX/releases

Obsidian

官网下载，进入More playforms，选择Universal DMG（Intel and M1）
https://obsidian.md

Chrome

官网直接下载即可
https://www.google.cn/chrome/index.html

Postman

官网下载，选择Mac Apple Chip
https://www.postman.com/downloads/?utm_source=postman-home

VS Code

官网下载，选择other playforms，再选择.zip中的Apple silicon
https://code.visualstudio.com/#alt-downloads

Sequel Pro

官网点击下载即可，不区分Intel和M1
https://sequelpro.com
将工具拖进程序坞中

运行时会崩溃换成下面的
https://sequelpro.com/test-builds

iterm2

https://iterm2.com/downloads.html

IDEA社区版

官网下载，进入下载，点击.dmg选择.dmg(Apple Silicon)
https://www.jetbrains.com/zh-cn/idea/

Sublime text

官网，无m1版本
http://www.sublimetext.com/
